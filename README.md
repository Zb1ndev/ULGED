# Ultra Lightweight Graphical Editor ( ULGED )
ULGED is a Graphical Text / Code Editor written in C using Raylib.

## Controls
- Zoom In / Out : ```CTRL + '='``` ```CTRL + '-'```
- Scroll Up / Down : ```CTRL + 'UP'``` ```CTRL + 'DOWN'``` or ```SCROLL_WHEEL``` 
- Open Command Palette : ```CTRL + 'P'```
